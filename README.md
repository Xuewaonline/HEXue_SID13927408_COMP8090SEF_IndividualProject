## 🎯 Task 1: SmartPET 🐱🐶 - Pet Shop Management System

### Description
SmartPET is a console-based Python application that demonstrates core Object-Oriented Programming concepts through a practical pet store management system.

### Core Features
| Feature | Description |
|---------|-------------|
| **Pet Inventory Management** | Add, view, and manage pets with details (name, age, price, breed, type) |
| **Customer Tracking** | Maintain customer records with contact info and purchase history |
| **Sales Processing** | Sell pets to customers with automatic inventory updates and revenue tracking |

## 🎯 Task 2: Self-Study - Data Structures & Algorithms

### Description
This task involves self-study and implementation of a fundamental data structure (Graph) and a sorting algorithm (Cocktail Sort).

### Topics Covered

#### 1. Data Structure: Graph
Implementation of three graph types with core operations:
- **Undirected Graph** - Bidirectional edge relationships
- **Directed Graph** - One-way edge relationships  
- **Weighted Graph** - Edges with associated weights/costs

#### 2. Algorithm: Cocktail Sort
Implementation and analysis of the **Cocktail Sort algorithm**
